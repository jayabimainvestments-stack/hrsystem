# HRMS පද්ධතිය වෙනත් පරිගණක වල භාවිතා කරන ආකාරය

ඔබගේ ශාඛාවේ වෙනත් පරිගණක වලින් මෙම HRMS පද්ධතියට පිවිසීමට පහත පියවර අනුගමනය කරන්න.

### 1. Firewall සකසන්න
මෙය එක් වරක් පමණක් සිදුකිරීම ප්‍රමාණවත්ය.

1. `open_ports.bat` ගොනුව සොයාගන්න (මෙම ෆෝල්ඩරයේ).
2. එම ගොනුව මත Right-click කර **"Run as administrator"** තෝරන්න.
3. කළු පැහැති තිරයක් (Console) විවෘත වී "Ports opened successfully" ලෙස දිස්වනු ඇත.

### 2. පද්ධතිය ක්‍රියාත්මක කරන්න
1. සුපුරුදු පරිදි `start_servers.bat` ගොනුව ධාවනය කරන්න.

### 3. වෙනත් පරිගණක වලින් පිවිසෙන්න
වෙනත් ඕනෑම පරිගණකයක වෙබ් බ්‍රව්සරය (Chrome, Edge, Firefox) විවෘත කර පහත ලිපිනය (URL) ඇතුලත් කරන්න:

**http://10.47.219.33:5173**

### ගැටළු විසඳීම (Troubleshooting)

**පද්ධතියට සම්බන්ධ විය නොහැකිනම්:**
1. පරිගණක දෙකම එකම ජාලයකට (Wi-Fi හෝ LAN) සම්බන්ධ වී ඇති බව තහවුරු කරගන්න.
2. ප්‍රධාන පරිගණකයේ `start_servers.bat` ක්‍රියාත්මක වී ඇති බව තහවුරු කරගන්න.
3. ප්‍රධාන පරිගණකයේ IP ලිපිනය වෙනස් වී ඇත්දැයි පරීක්ෂා කරන්න:
   - `Windows Key + R` ඔබන්න.
   - `cmd` ලෙස Type කර Enter කරන්න.
   - `ipconfig` ලෙස Type කර Enter කරන්න.
   - `IPv4 Address` අගය සොයාගෙන නව IP ලිපිනය භාවිතා කරන්න.
