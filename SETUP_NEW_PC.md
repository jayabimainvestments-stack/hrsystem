# HR Management System Setup Guide (New PC)

This guide explains how to set up the HR Management System on a new computer while preserving all current data.

## 1. Prerequisites
Ensure you have the following installed:
- **Node.js** (v18 or higher)
- **PostgreSQL** (v14 or higher)

## 2. Directory Structure
Copy the entire `HR PACEGE` folder to your new computer's drive (e.g., `D:\HR PACEGE`).

## 3. Database Restoration
1. Open **pgAdmin** or **SQL Shell (psql)**.
2. Create a new database named `hr_db`.
3. Execute the SQL commands from `hr_db_backup.sql` to restore the schema and data.
   - Using psql: `psql -U postgres -d hr_db -f hr_db_backup.sql`

## 4. Backend Setup
1. Navigate to the `backend` folder.
2. Open `.env.example` and rename it to `.env`.
3. Update the `DATABASE_URL` with your local PostgreSQL credentials:
   `DATABASE_URL=postgres://postgres:YOUR_PASSWORD@localhost:5432/hr_db`
4. Open a terminal in the `backend` folder and run:
   ```bash
   npm install
   npm run dev
   ```

## 5. Frontend Setup
1. Navigate to the `frontend` folder.
2. Open a terminal and run:
   ```bash
   npm install
   npm run dev
   ```
